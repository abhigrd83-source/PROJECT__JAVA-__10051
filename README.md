# PROJECT__JAVA-__10051
Online Food Delivery System

This is a simple Java project that simulates an online food ordering system. In this program, a user can view the menu, select food items, enter quantity, and get the final bill.

The system handles basic operations like taking input from the user, storing selected items, and calculating the total amount along with delivery charges and tax.

---

Features

- Show food menu
- Select items with quantity
- Add multiple items
- Calculate subtotal
- Apply delivery charge
- Apply tax
- Display final order summary

---

How to Run

1. Open the project in any Java IDE
2. Compile the program
3. Run the Main.java file
4. Follow the instructions on screen

---

Example

Menu:

1. Burger - ₹100
2. Pizza - ₹300

Order Summary:
Burger x2 = ₹200
Pizza x1 = ₹300
Subtotal: ₹500
Delivery Charge: ₹0
Tax (5%): ₹25
Total Amount: ₹525
